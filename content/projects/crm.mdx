## Context

Sales lived in spreadsheets and ad-hoc Slack threads. Leadership wanted one pipeline view that still felt fast on mediocre hotel Wi‑Fi.

## What I did

- Built a table layer with virtualized rows, column persistence, and bulk actions that never blocked the main thread.
- Added a lightweight “activity composer” with templates so reps logged meetings in seconds.
- Wrote integration tests around the worst historical regressions (timezone boundaries and filter state).

## Outcome

Higher data quality in the CRM, less context switching, and a component set reused in two later internal tools.
